# Eddystone lighthouse

[Eddystone lighthouse](https://en.wikisource.org/wiki/The_American_Cyclop%C3%A6dia_(1879)/Eddystone_Rocks) rendered as ASCII with my own C# tool.

This piece was generated in console (Powershell) variant.
The main console screenshots use 12 px Cascadia Mono at about 100 characters wide.
I also exported a 12 px Consolas version for this readme and a variable-width version for comparison.

No GenAI was used. The result comes from image preprocessing, thresholding, optional dithering and character matching.

```powershell
.\AsciiPhoto2 -v true -aj ".\Alphabets\Cascadia Mono 12px min.json" -tw 800 -pd true -sp false -fow 1 -of ".\Ascii art, Consolas 12px.md" -fom 1 -ob true -p "D:\Work\Ascii photo 2\Documents\Test images\AmCyc_Eddystone_Rocks_-_lighthouse.jpg" --auto false -ud true -cpw 170/50/0 -o 12/4
```
