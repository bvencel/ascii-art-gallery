This is a public showcase repository for my finished ASCII art pieces.

The purpose of this repo is to:

- Present the final artwork clearly
- Preserve the raw output behind each piece
- Show that the work is reproducible and not a black box

Each artwork folder may include:

- The source image
- Rendering parameters
- One or more generated `.md` files
- Final screenshots
- Comparison screenshots, such as original vs non-dithered vs dithered

## No GenAI

These pieces are not generated with GenAI.

They are produced with a custom rendering workflow based on image processing, optional dithering and character matching.

## Why this repo exists

This repo exists to support the public side of the project:

- Sharing finished pieces on social media
- Giving each post a stable home
- Letting people inspect the files behind the result
- Building a growing public gallery of ASCII work
