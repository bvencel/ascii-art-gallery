# Corinthian column

A Corinthian column sketch rendered as ASCII with my own C# tool.

This piece was generated in console (Powershell) variant.
The main console screenshots use 12 px Cascadia Mono at about 100 characters wide.
I also exported a 12 px Consolas version for this readme and a variable-width version for comparison.

No GenAI was used. The result comes from image preprocessing, thresholding, optional dithering and character matching.
